Super Mario Princess

Ferramentas Utilizadas

Desenvolvi um jogo inspirado na Princesa do Super Mario usando JavaScript, HTML e CSS. Neste jogo, os jogadores podem ajudar a Princesa a resgatar seus amigos e coletar itens especiais, enfrentando diversos desafios ao longo do caminho. Utilizei HTML para estruturar o jogo, CSS para estilizar os elementos e criar um visual atraente, e JavaScript para adicionar interatividade e lógica de jogo, como movimentação dos personagens e detecção de colisões. Este projeto demonstra minhas habilidades em desenvolvimento web, incluindo a criação de interfaces dinâmicas e responsivas, e minha paixão por criar experiências de jogo envolventes e divertidas.

MobilleApp Gym

Ferramentas Utilizadas

Desenvolvi uma aplicação mobile com TypeScript e React Native para criar uma tela de login para uma academia fictícia. O projeto visa proporcionar uma interface intuitiva e responsiva para os usuários acessarem suas contas de forma segura. As principais funcionalidades incluem autenticação de usuário, validação de formulário em tempo real, design responsivo, animações suaves e boas práticas de segurança. Utilizei TypeScript para adicionar tipagem estática, React Native para o desenvolvimento da interface mobile e Expo para facilitar o desenvolvimento e a execução do aplicativo.
